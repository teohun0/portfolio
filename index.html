How to add your own images (beginner guide)
  1. Create a folder named images beside index.html.
  2. Copy your photos into that folder, for example: cert1.jpg and group1.jpg.
  3. Find the image line you want to replace, e.g. src="https://placehold.co/...".
  4. Change it to your file path, such as src="images/cert1.jpg".
  5. Change the caption text under it if needed, then save and refresh your browser.
